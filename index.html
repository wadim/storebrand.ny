Storebrand.ny - Version 0.92

Web manual and form manual as of 2016-05-11

This version is not yet official and may still be subject to change without warning.

So, what's new in this version?

  • Upgraded Bootstrap from 3.3.5 to 3.3.6
  • Fix for fontfile SignaWeb Extralight (flawed line-height on Mac)
  • Cleaned up Gruntfile (+removed unused stuff)
  • All public/generated files are now in ./public (files are copied on save)
  • Fixed sourcemaps for JS+LESS
  • Most of our dependencies are now handled with NPM (remember to "npm install")
  • Some JavaScript code has been separated into its own files, for improved intuitivity
  • Production files are now named "stb-all.min.css" and "stb-all.min.js", and the javascript folder was renamed to js
  • Various changes in markup in order to support "universell utforming", WCAG usability requirements
  • Added p.header classes that can be used to keep the desired look of various text, without breaking h tag structure
  • Made a few font changes to existing header tags
  • Removed font aliasing in body

How to play

Here's the gzipped code you'll want to include to start playing with this version:

 <link rel="stylesheet" type="text/css" href="//elements.storebrand.no/storebrand.ny/0.92/css/stb-all.min.css">
 <script type="text/javascript" src="//elements.storebrand.no/storebrand.ny/0.92/js/stb-all.min.js"></script>
